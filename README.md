## 概要
- GetX + Dio + Setting UI + flutter_login を用いた、スーパーシンプルリポジトリです。

## 画面
- ログイン
- ホーム　　
- 設定 
- ナビゲーションバー

## 機能
- ログイン（モック）
  - 'test1@test.com': '12345',
  - 'test2@test.com': 'test1234',
- 画面遷移
- API通信

## ディレクトリ構成
lib  
∟ data : API通信  
∟ page : 各画面  
∟ routes : ルート  
∟ themes : テーマ
